# Under Evaluation

Tools being actively considered but not yet adopted.

## AeroSpace

[AeroSpace](https://github.com/nikitabobko/AeroSpace) is a tiling WM that operates at the app level rather than hooking into the macOS window server. It may handle Ghostty's native tab grouping without the window-jumping issue that [rules out Ghostty](why-not.md#ghostty), making a Ghostty migration possible.
