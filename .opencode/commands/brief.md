Create an executive brief for: $ARGUMENTS

Format:
- **1 page maximum** (300 words)
- **Opening line:** Why this matters (1 sentence)
- **Key points:** 3-5 bullets max
- **Recommendation:** Clear next action
- **Trade-offs:** Costs vs benefits (if applicable)

Use:
- Executive voice (decisive, no hedging)
- Active voice only
- Max 15 words per sentence
- Zero AI tells
- Radical brevity

If $ARGUMENTS is a file path, read it first.
If it's a topic, ask for key information needed.
